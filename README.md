A reusable Python pipeline for cleaning and preprocessing raw datasets. Built for Data Engineers and Analysts who need reliable, automated data quality workflows.

## Features

- Remove duplicate rows automatically
- - Handle missing values with configurable strategies
  - - Standardize and normalize column names
    - - Export cleaned data to CSV or JSON
      - - Generate data quality report

          ## How to Run

          1. Clone this repository:
          2. ```bash
             git clone https://github.com/MoisesVasconcelos10/data-cleaning-pipeline.git
             cd data-cleaning-pipeline
             ```

             2. Install dependencies:
             3. ```bash
                pip install pandas
                ```

                3. Run the pipeline:
                4. ```bash
                   python cleaning.py
                   ```
